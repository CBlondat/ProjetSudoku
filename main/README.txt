Cyprien BLONDAT

Projet : Sudoku

Objectif : Réaliser un sudoku sur console

Réalisation :
- Génération d'une grille 9*9
- Grille 9*9 correcte selon les règles du sudoku (les chiffres 1 à 9 sur chaque ligne, colonne, et mini-grilles)
- Affichage de la grille
- Possibilité de voir le nombre d'erreurs commises
- Possibilité d'afficher la solution
- Option pour quitter le jeu
